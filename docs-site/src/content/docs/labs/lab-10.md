## GOALS
- Know where and how to create different types of reports
- Understand the process of creating custom widgets
- Understand the difference between graphical reports and table reports
- Explore available reporting and scheduling options
- Learn how CTD presents information for different audiences
---
## TASKS
### Part One: Exploring Existing Reports

1. Navigate to **Reports > Reports Library**.
2. Review the reports that are already present within the system.
3. Examine several predefined reports and note:
   - The report name
   - The widgets included
   - The information being displayed
4. Open the report details and review any available configuration options.
---
### Part Two: Creating a Custom Widget
1. Navigate to **Visibility > Assets**.
2. Open the kebab menu (⋮) and select **Select Columns**.
3. Review the available columns and note the variety of asset information available for reporting.
4. Open the kebab menu again and select **Create Widget**.
5. Configure the widget:
   - Select a chart type.
   - Group by **Asset Criticality**.
   - Review any additional options available.
6. Save the widget.
---
### Part Three: Creating a Report
1. Navigate to **Reports > Reports Editor**.
2. Select **Add Widget**.
3. Add the custom widget created previously.
4. Add at least two predefined widgets.
5. Review the available widget categories and note the types of information that can be included in a report.
6. Select **Create**.
7. Review the available report options:
   - Report Name
   - Description
   - Scheduling Settings
   - Recurrence Options
   - Recipient Options
8. Create the report.
---
### Part Four: Exploring Report Scheduling
1. Navigate to **Reports > Reports Library**.
2. Locate the report you created.
3. Open the report details.
4. Review:
   - Scheduling settings
   - Report ownership
   - Delivery options
   - Available actions
5. Explore the options available for editing an existing report.

> Reports can only be emailed to users that exist within CTD. SMTP must be configured for email delivery to function.
---
### PART Five: Creating a Custom Table Report
1. Navigate to **Risks & Vulnerabilities > Insights**.
2. Open the kebab menu (⋮).
3. Select **Create New Table Report**.
4. Review the configuration options available.
5. Select a recurrence schedule.
6. Create the report.
7. In the proceeding pop up menu, click <u>Go to the Table Reports Library to view your new Report</u>.
8. Filter **Report Source** by **Custom**.
9. Locate the report you created.
10. Select the **Schedule Report** (🕒) icon and review the available configuration options.
---
### Part Six: Comparing Report Types
1. Review both the graphical report and table report created during this lab.
2. Compare:
   - Information displayed
   - Available customization options
   - Scheduling options
   - Intended use cases
3. Explore additional widget options and note:
   - Which CTD pages allow widgets to be created
   - Which types of data appear most frequently in reporting
